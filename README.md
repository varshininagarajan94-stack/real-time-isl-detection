# Real-Time Indian Sign Language Detection 🤖✋

## 📌 Project Description
This project is an AI-based system that detects Indian Sign Language gestures in real time and converts them into text. It helps deaf and mute people communicate easily with normal users.

## 🎯 Objective
- To recognize hand gestures using AI
- To convert gestures into readable text
- To improve communication for hearing-impaired individuals

## 🚀 Features
- Real-time gesture detection
- AI-based recognition system
- Converts sign language into text
- User-friendly interface

## 🛠️ Technologies Used
- Python
- OpenCV
- Machine Learning
- AI Models
